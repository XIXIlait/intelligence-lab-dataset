# Projet Titanic — Intelligence Lab by ECE
### Prédire la survie des passagers du RMS Titanic

Dépôt GitHub : https://github.com/XIXIlait/intelligence-lab-dataset

---

## Objectif

À partir des données réelles des passagers du Titanic, construire pas à pas un modèle capable de prédire si un passager a survécu ou non au naufrage du 15 avril 1912. Ce projet couvre l'ensemble du pipeline : chargement, nettoyage, visualisation, feature engineering, entraînement et analyse des résultats.

---

## Démarrage rapide

Si vous n'avez jamais utilisé Google Colab, commencez par regarder ce tutoriel :
**Tuto Google Colab :** https://www.youtube.com/watch?v=WaEbAtuRhM4

**Étape 1 — Ouvrir Google Colab**
Rendez-vous sur colab.research.google.com et connectez-vous avec un compte Google.

**Étape 2 — Importer le notebook**
Cliquez sur "Fichier" → "Ouvrir un notebook" → onglet "Importer", puis glissez-déposez le fichier `Projet_Titanic_Dataset.ipynb` téléchargé depuis le dépôt GitHub.

**Étape 3 — Exécuter les cellules**
Exécutez les cellules dans l'ordre, de haut en bas, en appuyant sur `Shift + Entrée` ou en cliquant sur le bouton ▶ à gauche de chaque cellule.

---

## Structure du notebook

Le notebook est composé de 9 sections qui suivent un pipeline complet de Machine Learning. Chaque cellule est étiquetée :

- Cellule bleue — EXECUTER : lancez la cellule sans rien modifier
- Cellule verte — A COMPLETER : complétez le code en suivant les commentaires guidés
- Cellule orange — MODIFIER : ajustez les paramètres pour expérimenter

---

## Bibliothèques utilisées

Toutes sont préinstallées sur Google Colab, aucune installation requise.

- pandas — manipulation de données
- numpy — calculs numériques
- matplotlib / seaborn — visualisation
- scikit-learn — modèle de Machine Learning
